## Feedback Command

The `/feedback` command allows you to send feedback, suggestions, bug reports, or feature requests directly to the developer.

### Usage

```
/feedback
```

### How to Use

1. Use `/feedback` to open the feedback interface
2. Click **"Give Feedback"** button
3. Fill out the modal:
   - **Feedback Type**: What type of feedback (bug reports, suggestions, feature requests, etc.)
   - **Content**: Your detailed feedback message
4. Submit your feedback

### Types of Feedback

You can send various types of feedback:

- **Bug Reports**: Report issues or errors you've encountered
- **Suggestions**: Share ideas for improvements
- **Feature Requests**: Request new features or commands
- **General Feedback**: Share your thoughts and opinions

### What Happens

- Your feedback is sent directly to the developer via webhook
- Your username and user ID are included for context
- The developer reviews all feedback regularly

### Tips for Good Feedback

- **Be specific**: Describe the issue or suggestion clearly
- **Include details**: Provide steps to reproduce bugs
- **Be constructive**: Explain what you'd like to see improved
- **Be patient**: The developer reviews feedback but may not respond immediately

### Privacy

- Your Discord username and ID are included with feedback
- This helps the developer understand context and follow up if needed
- Feedback is kept private between you and the developer

### Notes

- Feedback is always appreciated
- All feedback is reviewed, even if not all suggestions can be implemented
- Check `/roadmap` to see if your suggestions are being considered
