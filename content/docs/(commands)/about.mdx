## About Command

The `/about` command displays information about HoYoVista, including statistics, version information, and links to important resources.

### Usage

```
/about
```

### Information Displayed

- **Guild Count**: Total number of Discord servers using HoYoVista
- **Developer**: Bot creator (ScobbleQ)
- **Version**: Current bot version
- **Node.js**: Node.js version running the bot
- **Discord.js**: Discord.js library version
- **Uptime**: How long the bot has been running (relative time)

### Links Provided

- **Directory**: Link to Discord's application directory listing
- **GitHub**: Link to the bot's GitHub repository
- **Docs**: Link to the documentation website
- **Server**: Link to the support Discord server

### Use Cases

- Check bot status and uptime
- Verify you're using the latest version
- Access important links quickly
- Learn about the bot's technical details

### Notes

- Guild count updates in real-time
- Uptime resets when the bot restarts
- Version information comes from package.json
