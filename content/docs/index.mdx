## Welcome to HoYoVista

HoYoVista is a powerful Discord bot designed to enhance your HoYoverse gaming experience. Access all your HoYoLAB features directly from Discord, including daily check-ins, profile viewing, real-time notes, income tracking, and automatic code redemptions.

### Getting Started

1. **Invite the bot** to your Discord server using the [Invite Link](https://discord.com/oauth2/authorize?client_id=1267369071444492309)
2. **Register your account** using `/register`
3. **Link your HoYoLAB account** using `/hoyolink` (you'll need your HoYoLAB cookies)
4. **Start using commands** like `/check-in`, `/profile`, `/notes`, and more!

### Supported Games

HoYoVista supports the following HoYoverse titles:

- Genshin Impact
- Honkai: Star Rail
- Honkai Impact 3rd
- Zenless Zone Zero
